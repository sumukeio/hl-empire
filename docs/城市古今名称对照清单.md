# 中国城市古今名称对照清单

> 覆盖全国 **34 个省级行政区** 下的 **地级行政区**（含地级市、自治州、地区、盟）及 **直辖市**；并附 **台湾省** 现行县市级划分、**港澳** 特别行政区。
>
> **择名原则**：一地有多个历史名称时，取最经典、最耳熟能详、最具文学传播力者。
> 行政区划基准：**2024 年**（地级共 333 个，加 4 直辖市计 337）。

## 使用说明
- **收录条目**：361 条（地级 337 + 台湾 22 + 港澳 2）
